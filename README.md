# Mirror Master

Mirror Master 是一个纯浏览器、本地运行的可变尺寸网格镜像工具。图片不会上传到服务器，也不依赖后端、OCR 或外部图像服务。

## 使用流程

1. 上传一张包含规则方格的 PNG、JPEG 或 WebP 图片。
2. 应用会自动寻找主网格的外框，并根据重复边界推断列数、行数与正方形单元格尺寸。
3. 检查原图上的选区。选区只应覆盖需要镜像的主网格本体，不应包含标题、外围编号、图例、水印或空白边缘。
4. 如果自动识别不完整，可直接拖动选区，或拖动边缘与角点重新框选。应用会把选区限制在图片范围内，自动拟合等大的正方形单元格并重新推断整数行列，无需输入坐标、行列数或单元格尺寸。
5. 点击“生成镜像”查看结果；需要修改时点击“返回调整”。结果生成后可点击“下载结果 PNG”保存本地 PNG 文件。

## 镜像方式

镜像结果与原图尺寸完全相同。应用先复制完整原图，再仅处理选中的主网格本体：每一行保持不变，每个单元格从源列移动到 `列数 - 1 - 源列`。单元格内部像素不会翻转或缩放，因此 H7、A17、B13 等内部标签仍保持正向可读；选区以外的所有像素保持不变。

重新生成、重置选区、更换图片或关闭页面时，应用会清理旧的结果下载地址。所有识别、镜像与 PNG 导出均在当前浏览器中完成。

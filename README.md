# Mirror Master

Mirror Master 是一个纯浏览器、本地运行的 Pixelanim 网格镜像工具。所有受支持图片的逻辑结构固定为 34 列 × 27 行；图片的自然像素尺寸、网格左上角位置和正方形单元格的整数像素边长可以不同。图片不会上传到服务器，也不依赖后端、OCR 或外部图像服务。

## 使用流程

1. 上传一张 Pixelanim PNG、JPEG 或 WebP 图片。
2. 应用按照已知的 34 × 27 逻辑结构自动寻找网格左上角和整数单元格尺寸，不会推断其他行列数。
3. 检查原图上的选区。选区只应覆盖需要镜像的主网格本体，不应包含标题、外围编号、图例、水印或空白边缘。
4. 如果自动识别不完整，可直接拖出选区，或拖动整个选区、边缘与角点进行校正。移动只改变网格左上角；调整大小只改变整数单元格尺寸，并始终重新生成完整的 34 × 27 网格。选区会被限制在图片自然像素范围内。
5. 点击“生成镜像”查看结果；需要修改时点击“返回调整”。结果生成后可点击“下载结果 PNG”保存本地 PNG 文件。

## 镜像方式

镜像结果与原图尺寸完全相同。应用先复制完整原图，再逐一处理 27 行 × 34 列、共 918 个单元格（包括空白单元格）。每个完整的 `cellSize × cellSize` 单元格从源列移动到 `33 - 源列`，源尺寸和目标尺寸完全相同。单元格内部像素不会翻转、缩放或重建，因此 H7、A17、B13 等内部标签仍保持正向可读；选区以外的所有像素保持不变。

重新生成、重置选区、更换图片或关闭页面时，应用会清理旧的结果下载地址。所有识别、镜像与 PNG 导出均在当前浏览器中完成。

# Mirror Master

Mirror Master 是一个纯浏览器、本地运行的 34 × 27 网格镜像工具。图片不会上传到服务器。

## 使用流程

1. 上传一张 PNG、JPEG 或 WebP 图片。
2. 等待应用自动识别并显示 34 × 27 网格；如果未识别成功，直接在图片上拖出网格区域。
3. 在原图上拖动整个选区，或拖动边缘、角点调整大小。应用会自动保持 34:27 比例并生成等大的正方形单元格，无需输入坐标或单元尺寸。
4. 点击“生成镜像”，在同一工作区切换查看“镜像结果”；需要修改时点击“返回调整”。

## 镜像方式

镜像结果与原图尺寸完全相同。应用先复制完整原图，再把选中区域内的 34 × 27、共 918 个网格单元按列移动到镜像位置。每个单元内部的像素不会翻转或缩放，因此 H7、A17、B13 等内部标签仍保持正向可读；选区外的所有像素保持不变。
